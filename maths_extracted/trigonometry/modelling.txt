Modelling with
trigonometric functions
How do I model with trigonometric
functions?
Various real-life situations can be modelled using
trigonometric functions
You need to be able to interpret the equations used in the
model
If you need to identify maximum or minimum values of a
formula, remember the bounds of the sin and cos
functions:
-1 ≤ sin x ≤ 1
-1 ≤ cos x ≤ 1
